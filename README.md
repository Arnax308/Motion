# 🚀 Motion Pro (The Ultimate Lock-In App)

Bruh, if you ain't tracking your tasks with points, you're literally just doing side-quests. Welcome to **Motion Pro**, the ultimate productivity app that gamifies your entire life so you actually get stuff done. Built in **Python** and **Flet**, it turns grinding into an art form. No cap.

## ✨ Features (The Sauce)

- **📈 Productivity Points (PP) System:** Earn points for finishing tasks. Finish early? Bonus points. Miss a deadline? Say goodbye to your PP.
- **🔥 Streaks & Titles:** Start as a "Casual Rookie 🐣" and grind your way up to "Life Prodigy 🌟". Don't break the streak though.
- **📅 Dynamic Deadlines:** Reschedule tasks if you're feeling cooked, but be warned—you only get half the PP.
- **📊 Goated Analytics:** Heatmaps, category success rates, and monthly breakdowns so you know exactly where you're slacking.
- **🏷️ Categories:** Group your tasks into Work, Study, Exercise, etc., and see which part of your life is booming or busting.

## 🛠️ Stack

- **Python:** Clean, readable, W language.
- **Flet:** Building sleek UI without HTML/CSS. 
- **JSON:** Keeping your data local because your life is your business.

## 💻 How to run this

1. **Clone the repo** (if you haven't already):
   ```bash
   git clone https://github.com/Arnax308/Motion.git
   cd Motion
   ```

2. **Install dependencies**:
   ```bash
   pip install -r requirements.txt
   ```
   *(Just Flet, but do it anyway)*

3. **Fire it up**:
   ```bash
   python main.py
   ```
